
Prerequis
1-JDK8 ou une version superieur
2-Envirennement de developement eclipse
3-Avoir maven installer sur la machine
4-Avoir installer git et github
Execution
1-clonnez le projet et enregistrer le dossier 
2-importez le projet dans eclipse en projet maven 
3-allez a la classe DemoTest.java 
5-faire un clic droit et choisir run configuration
6-choisir junit5 et executer le programme
